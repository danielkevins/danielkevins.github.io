Job Applicant Passing Prediction Using the Naive Bayes Method

Summary

This project contains a website to predict how big the chances are of being accepted by prospective employees based on previous data, which data comes from employees who have been accepted and employees who have been rejected, so that the data is used as a reference to become training data for this naive bayes method.

Method & Algorithm

I use the Naive Bayes algorithm to classify data to predict the passing of job applicants. Naive Bayes is a classification method using probability and statistical methods proposed by British scientist Thomas Bayes.
The Naive Bayes algorithm predicts future opportunities based on previous experiences, so it is known as Bayes' Theorem. The main characteristic of this Naive Bayes Classifier is the very strong assumption (naive) of the independence of each condition / event.
